# Ejercicio
Automatización API PetStore "https://petstore.swagger.io/v2/pet"
Objetivo Agregar Mascota - Consultar - Modificar - Eliminar

Consideraciones:
## Requisitos

Para ejecutar el proyecto es necesario tener instalado lo siguiente:

  - Maquina local con el sistema operativo  Windows
  - JDK versión 11 (debe estar en la variable de entorno)
  - Intellij-Idea Comunity o Ultimate 
  - Gradle 3.3.10 (debe estar en la variable de entorno)



## Ejecucion
- Para ejecutar el proyecto
        1. Abrir el IDE IntellijIdea
        2. Ingresar en la opción File -> Open y ubicarse en el proyecto (abrir)
        3. Esperar unos minutos para que se descargen las despendencias (gradle)
        4. Ubicarse y ejecutar la clase ApiRunner.java (src\test\java\features\ApiRunner.java)
            - La clase ejecutara los scenarios de forma ordenada según el ejercicio realizado 
            - ## Reporte Karate: Una vez finalizada la ejecución se crea el informe propio de Karate, para visualizarlo 
		ubicarse en la ruta build\karate-reports\karate-summary.html
